The **Audience** section answers the most basic questions about a site's reach. How many people visited? Where did they come from? How many of them came back? Every other section in the Platform builds on the audience numbers, so this is the right place to start when getting oriented.

### What this section covers

| Page | What it covers |
|------|----------------|
| [User Acquisition](./acquisition) | How the Platform measures where audiences come from — channels, attribution windows, UTM tags |
| [Traffic Channel](./traffic-channel) | The standard channel taxonomy (Organic Search, Direct, Referral, Paid, Organic Social, Email, Other) |
| [Unique Visitors](./unique-visitors) | The deduplicated count of distinct users in a window |
| [DAU](./dau) | Daily Active Users — unique users with at least one meaningful action in a day |
| [MAU](./mau) | Monthly Active Users — unique users in a rolling 30-day window |
| [Retention](./retention) | Returning users, cohort curves, and the DAU/MAU stickiness ratio |

### How to read the section

Acquisition tells you where audiences originate. Audience size — Unique Visitors, DAU, MAU — tells you how large that audience is and how it's split by time window. Retention closes the loop by reporting how much of that audience comes back. Start with [Acquisition](./acquisition) for the conceptual overview, then drill into the metric pages as needed.
